# SEN3301 - 1st and 2nd Projects

This repository contains two mini-assignments (projects) developed for the SEN3301 course. These 2D foundational assignments serve as stepping stones for the main game project.

## 1. Mountain Background (1st Project)

- **Folder:** `mountain_background (1st Project)`
- Contains the Processing script (`project1_2201292.pde`) used to generate a 2D mountain landscape and environment.

## 2. Character Design (2nd Project)

- **Folder:** `character_design (2nd Project)`
- Includes the `characters` directory containing 2D character assets and a `RULES.docx` file detailing the specific guidelines and mechanics.
