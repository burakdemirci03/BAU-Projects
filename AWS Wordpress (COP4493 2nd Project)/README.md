# COP4493-AWS Academy Cloud Computing Fundamentals Project-2

This project demonstrates the deployment of a highly available and scalable WordPress website on AWS. The infrastructure includes a custom VPC, EC2 instances configured with NGINX and PHP, an RDS MySQL database, an Application Load Balancer (ALB), and an Auto Scaling Group (ASG).
